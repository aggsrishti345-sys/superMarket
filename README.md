## 📌 Description

**SuperMarket** is a full-stack e-commerce web application built using the **MERN stack (MongoDB, Express.js, React.js, and Node.js)**. The platform enables users to register, log in, browse products, manage carts, place orders, and view order history. Administrators can efficiently manage categories, products, and users through an admin dashboard.

This project demonstrates core concepts of full-stack web development including **CRUD operations, REST APIs, state management, authentication, and database integration**.

---

## ✨ Features

### 👤 User Features

* 🔐 User Registration & Login
* 🔑 Change Password
* 🛍️ Browse Products
* 🛒 Add Products to Cart
* 🧺 View Cart
* 📦 Place Orders
* 📜 View Order History

### 👨‍💼 Admin Features

* 📂 Category Management (Add, Edit, Delete)
* 📦 Product Management (Add, Edit, Delete)
* 👥 View Registered Users
* 🛠️ User Management
